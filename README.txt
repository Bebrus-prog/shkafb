Проект для отслеживания спортивного инвентаря для школьников/учителей. Поддерживается добавление инвентаря, закрепление его за пользователями, редакция списка инвентаря
Полное развёртывание на машине
1 установка NGINX и Gunicorn
2 конфигурация Gunicorn, проброс Django-приложения через SOCKS-протокол с его помощью.
3 создание конфигурации NGINX с указанием источника перенаправления (Gunicorn-сервер), запуск приложения на порте 443 и его открытие в системе. Конфигурация SSL-сертификата в NGINX.
4 проверка работоспособности приложения
5 создание daemon сервисов
